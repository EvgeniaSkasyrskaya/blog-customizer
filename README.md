# Проект «Блог. Кастомизация»

[Макет проекта](https://www.figma.com/file/FEeiiGLOsE7ktXbPpBxYoD/Custom-dropdown?type=design&node-id=0%3A1&mode=design&t=eXRJnWC6Xsuw0qR4-1)

### Стек

- React
- TypeScript
- Storybook
- CSS-переменные

### Цель проекта

Реализовать интерактивную панель кастомизации блога с использованием React-компонентов

### Задачи

- Изучение компонентов через Storybook
- Реализация формы из готовых UI-компонентов
- Управление состоянием — раздельное хранение данных формы и страницы
- Взаимодействие — применение стилей через CSS-переменные
- Механика — открытие/закрытие сайдбара, сброс настроек

### Функциональные требования

- При нажатии на «стрелку» открывается сайдбар с настройками, при повторном нажатии или клике вне сайдбар закрывается.
- При изменении настроек в сайдбаре они не применяются сразу.
- После нажатия на «применить» стили применяются к статье.
- При нажатии «сбросить» настройки в форме сбрасываются на начальные, которые были при открытии страницы, и стили применяются к статье.
- Настройки устанавливаются через CSS-переменные, которые уже есть в стилях и установлены в коде в дефолтные значения.

Запуск Storybook:

```
npm run storybook
```

Запуск линтера для стилей:

```
npm run stylelint
```

Запуск линтера:

```
npm run lint
```

Запуска форматтера:

```
npm run format
```
